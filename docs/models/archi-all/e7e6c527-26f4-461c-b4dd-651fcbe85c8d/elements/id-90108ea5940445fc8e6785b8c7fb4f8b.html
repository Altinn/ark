Meldingsrelaterte data ()
Meldinger kan sendes med referanser til _meldingsrelaterte data_, i stedet for å inkludere "alt" i hver melding. På denne måten kan hver enkelt melding holdes kort. Det blir opp til mottakerne å velge hvilke tilleggsdata som innhentes, ut fra kontekst og rettigheter.
