InterPlanetary File System (IPFS) ()
The InterPlanetary File System (IPFS) is a protocol and network designed to create a content-addressable, peer-to-peer method of storing and sharing hypermedia in a distributed file system. Essentially, IPFS seeks to improve on the current infrastructure of the web by changing the way data is accessed and transferred. Instead of requesting data from a specific location (as in HTTP), IPFS allows you to request data from anyone in the network who has it. The system gives each piece of data a unique fingerprint called a cryptographic hash, which enables the content to be found anywhere in the network. This distributed, decentralized nature of IPFS can potentially increase speed, reduce bandwidth usage, and improve resilience against network failures or censorship.
