Lenkede data ()
Linked Data" er en metode for strukturert og sammenkoblet presentasjon av data på nettet. Begrepet ble introdusert av Tim Berners-Lee, oppfinneren av World Wide Web, og refererer til prinsippene og standardene for å koble sammen data på en standardisert måte. Linked Data bruker unike identifikatorer (URI-er) for å identifisere og linke dataressurser. Dataene organiseres ved hjelp av RDF (Resource Description Framework), som gir en fleksibel måte å beskrive sammenhenger mellom dataelementer. Ved hjelp av Linked Data-prinsipper blir dataene mer maskinlesbare, søkbare og koblet sammen på tvers av forskjellige kilder og domener.
