Syncthing ()
Syncthing synchronizes files across multiple devices in real time, making your data available across your devices. Syncthing's primary feature is that data is not stored on a central server, like with Nextcloud and ownCloud, but is directly exchanged between devices. Syncthing is a peer-to-peer file synchronization solution, which means it doesn't require a central server. It uses its own Block Exchange Protocol to transfer files between nodes, with support for compression and delta updates to save bandwidth. Transfers are encrypted with TLS.
