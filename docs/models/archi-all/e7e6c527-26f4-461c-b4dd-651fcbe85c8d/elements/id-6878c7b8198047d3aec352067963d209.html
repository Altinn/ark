Referansearkitektur for datautveksling ()
Referansearkitektur for datautveksling beskriver i detalj verdikjeden for datadeling og gir i hovedsak organisatorisk samhandlingsevne mellom datatilbyder og datakonsument. Rammeverket beskriver mønster for datautveksling og omfatter alle former for flyt av data og hendelser (eller notifikasjoner) som gir teknisk samhandlingsevne.
