FileSender ()
FileSender is an open source web-based file transfer solution that is designed for sharing large files securely and efficiently. FileSender includes features such as encryption, user authentication, and expiration dates, which can help to ensure the security and privacy of shared files. FileSender is designed to transfer very large files and supports transfers up to several terabytes in size. It uses a web server and a database to store metadata, while the actual files are stored on a configured storage backend. File transfer is done via HTTP(S), with support for chunked uploads.
