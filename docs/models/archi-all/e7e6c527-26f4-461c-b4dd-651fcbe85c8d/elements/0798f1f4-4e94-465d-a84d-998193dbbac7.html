In EIRA, the requirement of concern is the interoperability requirement. An interoperability requirement is defined as a statement of an interoperable need that must be realized by a system. In EIRA, this statement is a High-level description of a to-be implemented interoperability aspect. An interoperability requirement adds granularity to an interoperability aspect
