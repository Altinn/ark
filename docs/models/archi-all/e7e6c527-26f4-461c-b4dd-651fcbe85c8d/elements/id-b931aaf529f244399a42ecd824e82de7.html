Referansearkitektur IAM ()
Referansearkitektur for IAM beskriver mønstre og beste praksis for identitet- og tilgangsstyring. Dette omfatter blant annet bruken av fellestjenester på IAM-området og beskrivelser av tillitskjeder som inngår i økosystemet for samhandling. Første del av referansearkitekturen for IAM som skal leveres i løpet av 2020 består av å:  Utarbeide en referansearkitektur og beskrivelse av hvordan man kan håndtere identitet for spesifikke systemer og entiteter/under-entiteter uten eget organisasjonsnummer.  Utarbeide en veileder i hvordan man kan identifisere en slik enhet. Referansearkitektur for IAM gir organisatorisk, semantisk og teknisk samhandlingsevne.
