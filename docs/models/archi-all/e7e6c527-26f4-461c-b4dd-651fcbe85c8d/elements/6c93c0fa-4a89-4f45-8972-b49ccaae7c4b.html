Det skal være mulig å lage gjenbrukbare ressurser, som: - webkomponenter (f.eks. i React) - Layouts - Look&feel - Tekster og oversettelser - Kodelister - Logikk (f.eks gjennom C#, TypeScript, WebAssembly) - Datamodeller (tilsv. Seres) - API-kall Å lage nye web componenter og layouts vil typisk være noe som tekniske utviklere gjør i kode-editorer. Det å sette disse sammen og konfigurere dem, er noe alle skal kunne gjøre.
